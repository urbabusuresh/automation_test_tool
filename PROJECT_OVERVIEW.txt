QUICK START:
------------
1. Install dependencies:
   $ pip install -r requirements.txt

2. Start the server:
   $ python app.py
   # Or use: ./run.sh

3. Open browser:
   http://localhost:5000

4. Record a test:
   • Go to "Record Test" tab
   • Enter test name and URL
   • Click "Start Recording"
   • Add actions
   • Click "Stop & Save"

5. Execute test:
   • Go to "Execute Tests" tab
   • Select tests
   • Click "Execute"
   • View results in "Test Results" tab

Example 1: Login Test

1. Start recording: "Login Test", "https://example.com/login"
2. Record actions:

   • Click: #login-button

   • Assertion: .welcome-message → "Welcome"

Example 2: Form Submission
--------------------------
1. Start recording: "Contact Form", "https://example.com/contact"
2. Record actions:
   • Input: #name → "John Doe"
   • Input: #email → "john@example.com"
   • Input: #message → "Test message"
   • Click: #submit-button
   • Assertion: .success-message → "Message sent"
3. Stop & Save
4. Execute test

Programmatic Usage:
-------------------
from recorder import ActionRecorder
from executor import TestExecutor

# Record
recorder = ActionRecorder()
recorder.start_recording("https://example.com", "My Test")
recorder.record_click("#button", "css")
test_path = recorder.stop_recording()

# Execute
executor = TestExecutor()
result = executor.execute_test_case(test_path)
print(f"Status: {result['status']}")
